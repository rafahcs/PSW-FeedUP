# FeedUP 
🚧Projeto em refatoração

## Propósito
Com o intuito de resolver as dores de um projeto desorganizado, sem planejamento e gerência eficiente, surge uma solução de gestão e organização. 
FeedUP é um software web para gerenciamento de ciclos de revisão em equipes de projeto. 
Os ciclos são workflows de contendo metas, tarefas e avaliações que podem ser revistos durante todo o tempo de realização do projeto.
- Gestores:
  - criam ciclos de revisão;
  - gerenciam as atividades dos participantes;
- Membros de equipe:
  - participam realizando as atividades;
  - recebem o feedback de seus gestores;

## Frontend com React e Redux
📁Frontend 
1. instale as dependêcias:
  ```
    npm i
  ```
2. instale o json-server:
  ```
  npm install -g json-server
  ```
3. rode em terminais separados as duas instruções:    
  ```
  npx json-server --watch db.json --port 3001
  ```
4. rode o software:
  ```
  npm run dev
  ```

## Backend com Node,Express,MongoDB,JWT
📁Backend
